int doj::alphanum_comp ( const lT &  left,
const rT &  right 
)

Compare left and right with the same semantics as strcmp(), but with the "Alphanum Algorithm" which produces more human-friendly results.

The classes lT and rT must implement "std::ostream operator<< (std::ostream&, const Ty&)".

Returns
negative if left<right, 0 if left==right, positive if left>right.
